# JavaScript Tutorial

A beginner-friendly JavaScript learning project with step-by-step exercises and small web demos.

## ✅ Project Overview

- Each lesson is in its own numbered folder (`01_Variable`, `02_Arithmetic`, etc.).
- Inside each lesson, `Website/` holds a simple HTML/CSS/JS demo for hands-on learning.
- Topics include variables, operators, control flow, loops, functions, arrays, objects, classes, and more.

## 📁 Folder Structure

- `01_Variable/Website/`
- `02_Arithmetic/Website/`
- ...
- `45_Nested_object/Website/`

Each lesson typically includes:
- `index.html` — basic page and input/output area
- `index.js` — JavaScript logic demonstrating the lesson concept
- `style.css` — optional styling for the demo

## ▶️ How to run

1. Open the folder in VS Code or your editor.
2. Open any lesson `Website/index.html` in browser (double-click or use a live-server extension).
3. Change code in `index.js` and refresh browser to test behavior.

## 🧠 Topics covered (quick list)

- Variables (`let`, `const`)
- Number math and `Math` object
- User input and type conversion
- Conditionals (`if`, `switch`, ternary)
- Loops (`while`, `for`)
- Functions and scope
- Arrays and methods (`forEach`, `map`, `filter`, `reduce`)
- Objects, nested objects, destructuring
- OOP with constructors, classes, `this`, `super`, static, getters/setters
- Random utilities (dice, password, number guessing)

## 💡 Usage

- Follow the numbers in order to progress from fundamentals to intermediate concepts.
- Use the exercises as starter code for your own JS practice projects.
- Reference MDN docs when you want formal definitions.

## 📦 Contributions

- Add new lessons in new numbered folders e.g. `46_NewTopic/Website/`.
- Keep filenames consistent (`index.html`, `index.js`, `style.css`).
- Submit a PR with a short description and what the lesson demonstrates.
